This version of Robotech: Battlecry is exactly the same as the standard edition, but it comes in a limited edition box with the following extras:

1. Robotech: Battlecry Game Soundtrack
2. Robotech: Battlecry t-shirt featuring "Skull One"
3. Five collectible art cards
4. Commemorative lenticular motion card
5. Robotech: Defense Force dog tag (individually numbered to note edition number of the set)
