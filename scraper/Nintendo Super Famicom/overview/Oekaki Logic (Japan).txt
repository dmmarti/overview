OEKAKI LOGIC IS NONOGRAM/PICROSS GAME FOR SUPER FAMICOM. THE GAME FEATURES 300 STANDARD PUZZLES WHICH START AT 5X5 AND GET AS LARGE AS 30X30, SEPARATED INTO FOUR DIFFICULTY TIERS. IT ALSO FEATURES A MODE WHERE PLAYERS CAN CREATE THEIR OWN PUZZLES AND SAVE THEM TO THE CART, AND A MODE WHERE TWO PLAYERS SIMULTANEOUSLY COMPLETE A SINGLE PICTURE, EARNING POINTS FOR EACH SUCCESSFUL "YES" AND "NO" SQUARE TO CREATE A COMPETITIVE ELEMENT.
