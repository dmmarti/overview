Match is a matching game written in GFA Basic where you have to match stone tiles so that you could draw a line between them. Two similar stones can be removed if they are:Adjacent Connect-able via a line Connect-able via a line with one edge Connect-able via a line with two edges The game has three game levels:Level 1, Stage 1-11: Easy to difficult Level 2, Stage 1-4: Predefined levels Level 3, Stage 1-?: Changing stone-sets, random sizes The game runs on time and when the time is up it's game over
