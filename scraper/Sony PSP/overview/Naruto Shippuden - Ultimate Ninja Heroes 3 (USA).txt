Naruto : Ultimate Ninja Heroes 3 est un jeu de combat/aventure sur PSP. Outre le mode Versus, un mode Histoire vous permet de vous battre aux côtés de deux partenaires contre plusieurs adversaires, d'améliorer vos jutsus, et d'effectuer des techniques combinées avec un équipier. De plus, vous avez le choix entre 51 personnages pour jouer contre un ami.
