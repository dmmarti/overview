Game Master is a cheat cartridge produced by Konami for the MSX computer. Generally, when inserted into slot 1 it allows to modify a Konami game which should be inserted in slot 2. The user can change the number of lives, set starting stage and modify other parameters and even save progress (TAPE) in certain games. Aside from this functionality the cartridge includes the ability to make screenshots and save them in SRAM and to print screenshots to a MSX printer.
